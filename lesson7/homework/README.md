# Homework: Conditional GAN на MNIST

## Задание

Заполните пропуски в ноутбуке `conditional_gan_mnist.ipynb` и обучите модель для генерации цифр по заданным меткам.

## Что нужно сделать

### 1. Заполнить ячейки с кодом

В ноутбуке есть несколько ячеек с пропусками (`...`), которые нужно заполнить:

#### Ячейка 7: ConditionalGenerator
- Реализуйте метод `forward` для генератора
- Генератор должен принимать шум `z` и метки `labels`
- Преобразуйте метки в one-hot векторы и конкатенируйте с шумом
- Пропустите через полносвязные слои для генерации изображения 28x28

#### Ячейка 9: ConditionalDiscriminator  
- Реализуйте метод `forward` для дискриминатора
- Дискриминатор должен принимать изображение `img` и метки `labels`
- Разверните изображение в вектор, преобразуйте метки в one-hot
- Конкатенируйте и пропустите через полносвязные слои для классификации

#### Ячейка 11: ConditionalGAN.training_step
- Реализуйте логику обучения Conditional GAN
- Обучите дискриминатор на настоящих изображениях с правильными метками
- Обучите дискриминатор на сгенерированных изображениях
- Добавьте проверку на неправильные метки (wrong_labels loss)
- Обучите генератор, чтобы обмануть дискриминатор

### 2. Обучить модель и получить результаты

- Обучите модель на датасете MNIST
- Добейтесь генерации реалистичных цифр, соответствующих заданным меткам
- Визуализируйте результаты: сгенерированные изображения должны соответствовать меткам (0-9)

## Критерии успеха

- Модель успешно обучается (loss'ы сходятся)
- Генерируемые изображения выглядят как цифры
- **Генерируемые цифры соответствуют заданным меткам** (ключевое требование)
- Визуализация показывает правильное соответствие между метками и изображениями
